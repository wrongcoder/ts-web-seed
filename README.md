! ts-web-seed

This is a workflow in a can.

!! Usage

1. Clone
1. Have Node.js and Node Package Manager on path
1. `$ npm install -g grunt-cli`
1. `$ npm install`
1. `$ grunt init`
1. Import `build.gradle` into IntelliJ
1. `$ npm dev`
1. Edit, save, reload, repeat
1. `$ npm test`
1. `$ npm dist && scp build/dist.zip webhost:`
